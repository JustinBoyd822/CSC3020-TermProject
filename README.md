This repository contains the term project for CSC3020 - Modern Technologies 
at Madonna University. The project demonstrates practical application of 
modern development tools, workflows, and containerization technologies 
throughout the semester.

**Project Focus:** Exploring modern software development practices including:
- Version control with Git/GitHub
- Project management with Jira and Confluence
- Containerization with Docker
- CI/CD pipeline integration
- Agile development methodology
